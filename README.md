## üìÇ Project directory structure

> üëâ **REMEMBER**:
>
> -   Path starting wiht "`.`" _([dotfile])_ indicates that the file or folder
>     is supposed to be **hidden** by some Operating Systems.
>
> -   Path endind with "`/`" _(forward slash)_ indicates that this is a **folder**.
>
> -   Files which contains pattern "`...rc.<extension>`" at the end,
>     is a **[run control file]** - in other words, a configuration file.

[dotfile]: https://en.wikipedia.org/wiki/Hidden_file_and_hidden_directory
[run control file]: https://en.wikipedia.org/wiki/Run_commands
